Bienvenido a mi pagina HTML

HTML es un lenguaje con el que se escriben las paginas web. las paginas web pueden ser vistas por el usuario mediante un tipo de aplicacion llamada navegador

Este lenguaje nos permite aglutinar textos,sonidos e imagenes y combilarlos a nuestro gusto.

El HTML nos permite la introduccion de referencias o otras paginas por medio de los enlaces hipertextos.
